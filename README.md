
# QR-Code Generator

Generate QR for your website with a few clicks on your keyboard.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Madflows/qr-code-generator
```

Go to the project directory

```bash
  cd qr-code-generator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
